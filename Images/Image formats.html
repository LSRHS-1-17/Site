What are image formats?

  • PNG- A PNG is an image that has two types, PNG-8 and PNG-24. PNG-8 only has 256 colors. PNG-24 however has a much higher quality and manage transparent pixels better. They also have large file sizes.
  • JPEG- A JPEG is an image that can allow for creation of difficult products. There are also many colors to choose from when working with JPEGs.
  • GIF- A GIF has very low file size, however they only contain 256 colors.
